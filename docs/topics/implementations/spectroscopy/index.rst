.. _spectroscopy:

=============
Spectroscopy
=============
A collection of components that implement the sika framework in order to fit models to spectroscopic data using nested sampling.


..  rst-class:: column

:ref:`Atmospheric Models <atmospheres>`
-------------------------------------------------------
Provide and manipulate atmospheric (pressure-temperature and pressure-abundance) models for brown dwarfs. 

..  rst-class:: column

:ref:`Spectra and Forward Modeling <spectra>`
-------------------------------------------------------
Represent, manipulate, and forward-model spectra.

..  rst-class:: column

:ref:`Instruments <instruments>`
--------------------------------
Instrument-specific components for loading and pre-processing data.

..  rst-class:: column

:ref:`Sampling <spectral-sampling>`
--------------------------
Pre-defined samplers for single and n-component modeling.

:ref:`Utilities <spectral-utils>`
----------------------------------
A grab-bag of utilities.
